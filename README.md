### Основные особенности игры

- камера от первого лица
- ходьба
- прыжки
- бег
- выносливость
- здоровье
- мана
- каст заклинаний кнопками 0-9
- каст заклинаний голосом через распознавание речи
- мультиплеер
- мультиплеер лобби
- режим каждый за себя
- командный режим
- режим захвата флага
- объемный голосовой чат
- статус эффекты
- множители характеристик
- подбираемые бонусы на карте

### Возможности заклинаний

- нанесение урона
- нанесение урона с течением времени
- нанесение урона в радиусе
- определение свой/чужой
- расход маны
- анимация призыва
- анимация каста
- эхо - повторение без затрат маны
- время жизни
- применение физической силы при контакте
- проход сквозь стены
- создание препятствий
- создание зон на земле
- скорость полета
- режим луча
- автонаводка
- channeling
- количество снарядов
- время между снарядами
- появления при нескольких снарядах - одна точка, по дуге
- спавн эфекта при попадании

## В разработке

### Архетипы и заклинания

| Архетип    | Заклинание      | Описание                                                            |
|:-----------|:----------------|:--------------------------------------------------------------------|
| Pyromancer | Fireball        | Мощный огненный шар с уроном по области вокруг точки попадания.     |
|            | Scorching Ray   | Луч огня ближнего боя с непрерывным уроном на коротком расстоянии.  |
|            | Flame Dash      | Ускорение с оставлением горящей земли.                              |
|            | Ember Lash      | Захват близко стоящих противников с поджогом и тянущим эффектом.    |
| Ranger     | Air Bullet      | Быстрый снаряд воздуха, наносит высокий точечный урон.              |
|            | Ice Shards      | Снаряды, оставляющие замораживающие области.                        |
|            | Charged Shot    | Заряжающийся выстрел - увеличивается урон, проходит сквозь стены.   |
|            | Shadow Step     | Мгновенное перемещение на большое расстояние.                       |
| Magician   | Ball Lightning  | Самонаводящийся заряд, наносит урон и проходит сквозь стены.        |
|            | Toxic Sphere    | Облако яда с уроном с течением времени и замедлением.               |
|            | Frost Nova      | Взрыв холода вокруг мага, замораживающий и повреждающий врагов.     |
|            | Dirt Wall       | Возведение земляных стен, блокирующих врагов.                       |
| Keeper     | Force Field     | Щит, поглощающий входящие снаряды.                                  |
|            | Rune Trap       | Ловушка, наносящая урон при контакте.                               |
|            | Shockwave Totem | Тотем, выбрасывающий ударную волну с уроном и отталкиванием врагов. |
|            | Pain Mirror     | Бафф дающий рефлект урона и уменьшение входящего урона.             |
